Image Search Abstraction Layer description
  • User can get the image URLs, alt text and page urls for a set of images relating to a given search string:
    • https://image-search-rfprod.c9users.io/imagesearch/scifi
  • User can paginate through the responses by adding a ?offset=2 parameter to the URL:
    • https://image-search-rfprod.c9users.io/imagesearch/scifi?offset=10
  • User can browse recent queries for images like this:
    • https://image-search-rfprod.c9users.io/latest/imagesearch/
  • Example response for image search request:
    • [
      {"url":"http://image_url-1","snippet":"title 1","thumbnail":"http://thumbnail_url-1","context":"http://context_url-1"},
      {"url":"http://image_url-2","snippet":"title 2","thumbnail":"http://thumbnail_url-2","context":"http://context_url-2"}
      ]
  • Example response for recent queries request:
    • [
      {"when":"2016-01-26 18:06","term":"term 1"},
      {"when":"2016-01-26 18:06","term":"term 2"}
      ]
